[Planets by FORGE3D]

This package designed to work with both the Universal Rendering Pipeline (URP) and the Built-in Rendering Pipeline in Unity.

** Switching Rendering Pipelines

To switch between the URP and Built-in pipeline, follow these steps:

    -Import the 'Rendering Pipeline Template' package from 'Assets/FORGE3D/Planets/SRP Templates'.
    -Choose the appropriate rendering pipeline template based on your preference.

Please note that this package has been tested with Unity versions 2021.3 to 2023.1.3, 6000.0.25f1

** Color Space

Planets by FORGE3D is built for Linear color space. To ensure optimal results, make sure to switch the Color Space to Linear in your Unity project settings. Here's how:

    -Go to Edit -> Project Settings -> Player -> Other Settings.
    -Locate the Color Space option and set it to Linear.

For more information on Unity's color space settings, you can refer to the Unity Manual - Color Space. https://docs.unity3d.com/Manual/LinearRendering-LinearOrGammaWorkflow.html
